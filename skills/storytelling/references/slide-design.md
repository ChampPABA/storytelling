## Core Principles

### The Assertion-Evidence Model (Alley)
Every content slide has two parts:
1. **Assertion headline** — a full sentence (8-14 words) that states the slide's conclusion
2. **Visual evidence** — a chart, image, diagram, or photo that supports the assertion

This is research-backed (Penn State, p < .01): audiences understand and remember assertion-evidence slides significantly better than traditional topic-subtopic slides.

**Good:** "ต้นทุนซ่อนเร้นสูงถึง 1.2 ล้านต่อปีจากระบบเดิม"
**Bad:** "ต้นทุน" or "Cost Analysis"

### The Glance Test (Duarte)
Every slide must pass the **3-second test** — if someone glances at it for 3 seconds, they should understand the main point. If not, there's too much on it.

Rules:
- Max **30 words** per slide
- Max **5 elements** per slide (6+ increases cognitive load by 500%)
- **40%+ white space**

### Signal-to-Noise Ratio (Duarte + Reynolds)
Every element must answer: "Does this help communicate the message?"
- If yes → signal (keep it)
- If no → noise (remove it)

Remove: decorative logos, generic clip art, footer bars, unnecessary borders, redundant labels.

### Picture Superiority Effect (Reynolds)
Images are remembered far better than words, especially after 30+ seconds. Every slide should have a dominant visual element. Use one powerful image rather than multiple small ones.

### One Idea Per Slide
Never combine multiple concepts on a single slide. If you have 3 points, use 3 slides. Build complexity progressively across slides, not within a single slide.

## Slide Types Taxonomy

### Opening Slides

**Title / Cover**
- Presentation title, presenter name, date, organization
- Full-bleed image or bold typography
- Sets the visual tone for the entire deck
- Duration: 15-30 seconds

**Agenda / Outline**
- List of sections/topics to be covered
- Optional timing indicators
- Use only for presentations >10 slides
- Duration: 30-60 seconds

### Transitional Slides

**Section Divider**
- Bold visual break between major sections
- Full-bleed image or solid color with section title
- Gives audience a mental "chapter break"
- Duration: 10-15 seconds (just a visual pause)

**Quote / Testimonial**
- Impactful quote with source attribution
- Large text, minimal other elements
- Use to add credibility or emotional weight
- Duration: 30-60 seconds

### Content Slides

**Problem / Challenge**
- Presents the problem that needs solving
- Use data or imagery to show scale/severity
- Creates narrative tension
- Duration: 60-120 seconds

**Data / Statistics**
- One key insight per chart — not a data dump
- Headline states the conclusion, chart provides evidence
- Label clearly, annotate the key finding
- Use declarative chart titles: "Revenue grew 30%" not "Revenue (2024)"
- Duration: 120-180 seconds

**Comparison**
- Side-by-side evaluation (before/after, old/new, us/them)
- Consistent formatting across both sides
- Highlight differences with color
- Duration: 60-120 seconds

**Timeline / Process**
- Chronological events or step-by-step workflow
- Visual markers (icons, numbers, arrows) for each stage
- Keep descriptions short — details go in speaker notes
- Duration: 60-120 seconds

**Key Takeaway / Summary**
- Recap main points with visual recap
- Reinforce the One Big Idea
- Often uses the same visual language as earlier slides
- Duration: 60-120 seconds

### Closing Slides

**Reward**
- What the audience gains from acting
- Frame at personal, team, and broader levels
- Duration: 45-60 seconds

**Call to Action**
- Specific, concrete next step
- Clear contact info or follow-up mechanism
- Duration: 30-60 seconds

**Thank You / Q&A**
- Optional — some presenters skip this
- Include contact details if appropriate
- Duration: varies

## Layout Patterns

### Primary Layouts (most common)

**Split Layout (Left-Right)**
Image/visual on one side, text on the other. Matches natural reading flow. Best for: problem slides, comparison slides, solution slides.

**Full-Bleed**
Image fills entire slide, text overlaid. Maximum emotional impact. Best for: title slides, section dividers, quote slides, S.T.A.R. moments.

**Centered**
Content centered with generous margins. Clean and focused. Best for: single-number statistics, CTAs, key takeaways.

**3-Card Horizontal**
Three equal boxes in a row. Best for: comparing 3 options, showing 3 steps, Rule of Three content.

### Extended Layouts

**Bento Grid** — asymmetric boxes of different sizes, magazine-like feel
**Timeline** — horizontal or vertical progression with markers
**Hub-and-Spoke** — central concept with radiating branches
**Funnel** — wide to narrow, showing filtering/conversion process
**Pyramid** — hierarchical layers, top-down or bottom-up
**Comparison Matrix** — grid comparing features across options
**Dashboard** — multiple data widgets on one slide (use sparingly)

### Composition Rules

**Rule of Thirds**
Divide the slide into a 3x3 grid. Place key elements at intersection points, not dead center. Creates more dynamic, visually interesting slides.

**Visual Hierarchy**
Guide the eye: bold headline → key visual → supporting detail. Use size, color, and position to establish clear reading order.

**Z-Pattern / F-Pattern**
For text-heavy slides, content follows the natural eye movement: top-left → top-right → bottom-left → bottom-right.

## Sequencing and Pacing

### Slide Count by Duration

| Duration | Slides | Notes |
|----------|--------|-------|
| 5 min | 5-7 | Tight, one idea each |
| 10 min | 8-12 | Sweet spot for most talks |
| 15 min | 10-15 | Allow for interaction |
| 20 min | 15-20 | 1 min/slide average |
| 30 min | 20-25 | Reserve 5 min for Q&A |
| 45 min | 25-35 | Reserve 10 min for Q&A |

**The 80% Rule:** Plan content for 80% of allocated time. The remaining 20% absorbs transitions, pauses, and audience interaction.

### Pacing Rhythm

Alternate between **dense slides** (data, process, comparison) and **breathing slides** (quote, divider, full-bleed image). Never stack more than 3 dense slides in a row — insert a breathing slide to let the audience process.

```
Dense → Dense → Breathing → Dense → Dense → Dense(max!) → Breathing
```

### Narrative Arc Sequence

**For Pitch storyline:**
```
1. Title
2. Problem (what is)
3. Data supporting problem (what is)
4. Unexpected insight (shift ↕)
5. Divider
6. Solution (what could be)
7. How it works (what could be)
8. Proof/ROI (what could be)
9. S.T.A.R. moment (shift ↕)
10. Comparison old vs new
11. Reward
12. CTA (new bliss)
```

**For Explanation storyline:**
```
1. Title
2. Agenda
3. Concept 1 (simple)
4. Example/demo
5. Divider
6. Concept 2 (building on 1)
7. Example/demo
8. Divider
9. Concept 3 (most complex)
10. Example/demo
11. Synthesis (all 3 connected)
12. Key Takeaway
```

## Speaker Notes

### Script Length
- Speaking rate: ~125 words/minute
- Per slide: 125-300 words depending on slide duration
- Use keywords and phrases, not full written-out paragraphs
- Think bullet prompts, not a manuscript

### Cardinal Rule: Complement, Don't Duplicate
Reading and listening compete for the same cognitive channel. Images and audio don't.

- **Slide shows:** visuals, key numbers, charts, images
- **Speaker says:** context, stories, examples, connections, meaning

Never read the slide to the audience.

### Structure Per Slide
```
[Transition IN]  — 1 sentence connecting from previous slide
[Main content]   — the story, explanation, or argument
[Action cue]     — (point at chart), (pause 2 seconds), (ask audience)
[Transition OUT] — 1 sentence setting up the next slide
```

## Timing Reference

| Slide Type | Recommended Time | Notes |
|------------|-----------------|-------|
| Title | 15-30s | Brief intro |
| Agenda | 30-60s | Quick overview |
| Section Divider | 10-15s | Visual pause only |
| Problem | 60-120s | Build context |
| Data/Statistics | 120-180s | Explain the insight |
| Quote | 30-60s | Let the words land |
| Comparison | 60-120s | Walk through both sides |
| Timeline/Process | 60-120s | Step through stages |
| Key Takeaway | 60-120s | Reinforce main points |
| CTA | 30-60s | Clear and concise |
| Thank You/Q&A | Remaining time | Open floor |

### Typography

| Element | Size | Weight |
|---------|------|--------|
| Headline (assertion) | 28-44pt | Bold |
| Subtitle | 24-28pt | Regular |
| Body text | 18-24pt min | Regular |
| Caption/reference | 12-14pt | Light |

Font: Always sans-serif for slides (Arial, Helvetica, Calibri, or the presentation theme font). Maximum 2 font families per deck.

### Color

- Use 2-3 colors maximum plus neutrals
- Cool colors (blue, green) for backgrounds; warm colors (red, orange) for emphasis
- Minimum contrast ratio: 4.5:1 for normal text, 3:1 for large text
- Avoid red-green combinations (8% of men are colorblind)
